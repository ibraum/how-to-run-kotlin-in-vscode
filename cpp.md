- Check in terminal or CMD:

```bash
g++ --version
```
- Search and install these extensions:

- Create a file named `main.cpp` Paste this code:

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World!" << endl;
    return 0;
}
```

- Run with

```bash
g++ main.cpp -o app
./app
```

- Output

```terminal
Hello World!
```
